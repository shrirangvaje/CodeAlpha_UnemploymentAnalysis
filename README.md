# Unemployment Analysis using Python 

## Project Overview
This project analyzes unemployment trends in India using Python.
The analysis focuses on understanding patterns over time and the
impact of the COVID-19 pandemic on unemployment rates.

## Dataset
- File name: `Unemployment in India.csv`
- The dataset contains unemployment statistics across different
  regions of India over time.

### Features Used
- Date
- Region
- Estimated Unemployment Rate (%)
- Estimated Employed
- Estimated Labour Participation Rate (%)

## Tools & Technologies
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Analysis Performed
1. Loaded and explored the dataset
2. Cleaned and preprocessed the data
3. Analyzed unemployment trends over time
4. Visualized unemployment rate patterns
5. Studied the impact of COVID-19 on unemployment

## Conclusion
The analysis shows a noticeable rise in unemployment during the
COVID-19 period, highlighting the significant economic impact of
the pandemic on employment in India.
